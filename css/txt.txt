Welcome to   РђЊ рЃАрЃбрЃўрЃџрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃАрЃљрЃЏрЃДрЃљрЃарЃЮрЃерЃў
рЃарЃљ рЃљрЃарЃўрЃА  ?
  (Cascading Style Sheets) рЃљрЃарЃўрЃА рЃћрЃюрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃўрЃА (рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА) рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. HTML рЃњрЃЋрЃљрЃФрЃџрЃћрЃЋрЃА рЃЋрЃћрЃЉ-рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА, рЃ«рЃЮрЃџрЃЮ   РђЊ рЃЏрЃўрЃА рЃцрЃЮрЃарЃЏрЃљрЃА, рЃцрЃћрЃарЃА, рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃљрЃА. рЃЏрЃљрЃарЃбрЃўрЃЋрЃљрЃЊ рЃарЃЮрЃЏ рЃЋрЃЌрЃЦрЃЋрЃљрЃЌ, HTML рЃљрЃарЃўрЃА рЃАрЃљрЃ«рЃџрЃўрЃА рЃЕрЃЮрЃюрЃЕрЃ«рЃў, рЃ«рЃЮрЃџрЃЮ   РђЊ рЃЏрЃўрЃАрЃў рЃўрЃюрЃбрЃћрЃарЃўрЃћрЃарЃў рЃЊрЃљ рЃћрЃЦрЃАрЃбрЃћрЃарЃўрЃћрЃарЃў.

рЃарЃљрЃбрЃЮрЃЏ рЃљрЃарЃўрЃА   рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў?
рЃЊрЃдрЃћрЃЋрЃљрЃюрЃЊрЃћрЃџ рЃЋрЃћрЃЉ-рЃЊрЃўрЃќрЃљрЃўрЃюрЃерЃў рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃћрЃцрЃћрЃЦрЃбрЃў рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃўрЃљ. рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃћрЃЉрЃА рЃАрЃБрЃарЃЌ, рЃарЃЮрЃЏ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃў рЃўрЃДрЃЮрЃА рЃљрЃарЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃцрЃБрЃюрЃЦрЃфрЃўрЃБрЃарЃў, рЃљрЃарЃљрЃЏрЃћрЃЊ рЃћрЃАрЃЌрЃћрЃбрЃБрЃарЃљрЃЊ рЃЏрЃўрЃЏрЃќрЃўрЃЊрЃЋрЃћрЃџрЃў рЃЊрЃљ рЃљрЃЊрЃЋрЃўрЃџрЃљрЃЊ рЃљрЃдрЃАрЃљрЃЦрЃЏрЃћрЃџрЃў.   рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃерЃћрЃЏрЃЊрЃћрЃњрЃерЃў:

рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃАрЃбрЃўрЃџрЃўрЃќрЃљрЃфрЃўрЃљ (рЃцрЃЮрЃюрЃбрЃћрЃЉрЃў, рЃцрЃћрЃарЃћрЃЉрЃў, рЃќрЃЮрЃЏрЃћрЃЉрЃў)

рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ (grid, flexbox)

рЃцрЃЮрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃЕрЃљрЃарЃЕрЃЮрЃћрЃЉрЃўрЃА рЃЊрЃўрЃќрЃљрЃўрЃюрЃў

рЃарЃћрЃАрЃърЃЮрЃюрЃАрЃўрЃБрЃарЃў рЃЊрЃўрЃќрЃљрЃўрЃюрЃў (рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃћрЃЉрЃќрЃћ рЃљрЃЊрЃљрЃърЃбрЃўрЃарЃћрЃЉрЃљ)

рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃљрЃарЃЊрЃљрЃЏрЃљрЃЋрЃљрЃџрЃў рЃћрЃцрЃћрЃЦрЃбрЃћрЃЉрЃў

рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА  ?
 -рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃАрЃљрЃЏрЃў рЃњрЃќрЃўрЃЌ:

Internal   РђЊ рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃгрЃћрЃарЃўрЃљ HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃерЃў <style> рЃбрЃћрЃњрЃўрЃА рЃерЃўрЃњрЃюрЃўрЃЌ.

External   РђЊ рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃфрЃљрЃџрЃЎрЃћ .  рЃцрЃљрЃўрЃџрЃерЃў рЃЊрЃљ HTML рЃцрЃљрЃўрЃџрЃерЃў рЃўрЃгрЃЮрЃЊрЃћрЃЉрЃљ <link> рЃбрЃћрЃњрЃўрЃЌ.

Inline   РђЊ рЃАрЃбрЃўрЃџрЃћрЃЉрЃў рЃЏрЃўрЃћрЃюрЃўрЃГрЃћрЃЉрЃљ рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа style рЃљрЃбрЃарЃўрЃЉрЃБрЃбрЃўрЃА рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ.

РюЁ   рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃћрЃЉрЃў рЃЊрЃљ рЃАрЃљрЃБрЃЎрЃћрЃЌрЃћрЃАрЃЮ рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃћрЃЉрЃў (Standards and Best Practices)

Introduction to Color
 
Color Рђћ рЃћрЃА рЃљрЃарЃўрЃА рЃћрЃарЃЌ-рЃћрЃарЃЌрЃў рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃў рЃЊрЃљ рЃЌрЃЋрЃљрЃџрЃерЃўрЃАрЃљрЃфрЃћрЃЏрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃЋрЃћрЃЉ рЃЊрЃўрЃќрЃљрЃўрЃюрЃерЃў. рЃАрЃгрЃЮрЃарЃў рЃцрЃћрЃарЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљ рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃюрЃљрЃЊ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃњрЃљрЃюрЃгрЃДрЃЮрЃЉрЃљ, рЃњрЃљрЃљрЃЏрЃљрЃарЃбрЃўрЃЋрЃЮрЃА рЃЎрЃЮрЃюрЃбрЃћрЃюрЃбрЃўрЃА рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ рЃЊрЃљ рЃњрЃљрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЮрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљ.

CSS-рЃерЃў рЃцрЃћрЃарЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃдрЃўрЃюрЃўрЃерЃюрЃЮрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃЮрЃарЃЏрЃљрЃбрЃўрЃЌ:

2. Hexadecimal (HEX) рЃцрЃћрЃарЃћрЃЉрЃў

}

  background-color: #f1c40f; /* рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў */
}
3. RGB рЃЊрЃљ RGBA рЃцрЃћрЃарЃћрЃЉрЃў
RGB (Red, Green, Blue): рЃцрЃћрЃарЃўрЃА рЃЊрЃљрЃАрЃљрЃ«рЃЏрЃљрЃарЃў рЃЎрЃЮрЃЏрЃърЃЮрЃюрЃћрЃюрЃбрЃћрЃЉрЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЋрЃљрЃарЃўрЃарЃћрЃЉрЃћрЃю 0-рЃЊрЃљрЃю 255-рЃЏрЃЊрЃћ.

css
Copy
 
h1 {
  color: rgb(255, 99, 71); /* рЃЌрЃ«рЃўрЃџрЃў рЃгрЃўрЃЌрЃћрЃџрЃў */
}
RGBA (Red, Green, Blue, Alpha): RGB-рЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ Alpha, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃцрЃћрЃарЃўрЃА рЃњрЃљрЃЏрЃГрЃўрЃарЃЋрЃљрЃџрЃЮрЃЉрЃљрЃА (0-рЃЊрЃљрЃю 1-рЃЏрЃЊрЃћ).

css
Copy
 
h1 {
  color: rgba(255, 99, 71, 0.5); /* 50% рЃњрЃљрЃЏрЃГрЃЋрЃўрЃарЃЋрЃљрЃџрЃЮрЃЉрЃљ */
}
4. HSL рЃЊрЃљ HSLA рЃцрЃћрЃарЃћрЃЉрЃў
HSL (Hue, Saturation, Lightness): рЃцрЃћрЃарЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃљрЃдрЃгрЃћрЃарЃўрЃџрЃўрЃДрЃЮ рЃарЃЮрЃњрЃЮрЃарЃф рЃбрЃЮрЃюрЃў (Hue), рЃАрЃўрЃЏрЃЎрЃЋрЃарЃўрЃЋрЃћ (Saturation) рЃЊрЃљ рЃюрЃљрЃЌрЃћрЃЉрЃљ (Lightness).

css
Copy
 
h1 {
  color: hsl(120, 100%, 50%); /* рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃцрЃћрЃарЃў */
}
HSLA (Hue, Saturation, Lightness, Alpha): рЃћрЃА рЃцрЃЮрЃарЃЏрЃљрЃбрЃў рЃљрЃњрЃарЃФрЃћрЃџрЃћрЃЉрЃА HSL-рЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃЊрЃљрЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃњрЃљрЃЏрЃГрЃўрЃарЃЋрЃљрЃџрЃЮрЃЉрЃљрЃА.

css
Copy
 
h1 {
  color: hsla(120, 100%, 50%, 0.3); /* рЃЏрЃгрЃЋрЃљрЃюрЃћ рЃцрЃћрЃарЃў 30% рЃњрЃљрЃЏрЃГрЃЋрЃўрЃарЃЋрЃљрЃџрЃЮрЃЉрЃўрЃЌ */
}
РюЁ рЃцрЃћрЃарЃЌрЃљ рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃћрЃЉрЃў
рЃЋрЃћрЃЉ рЃЊрЃўрЃќрЃљрЃўрЃюрЃўрЃА рЃЊрЃарЃЮрЃА рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃћрЃЉрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЋрЃљрЃюрЃўрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃЏрЃљрЃарЃЌрЃџрЃўрЃљрЃюрЃЮрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃЎрЃўрЃЌрЃ«рЃЋрЃўрЃА рЃЎрЃЮрЃЏрЃцрЃЮрЃарЃбрЃўрЃАрЃЌрЃЋрЃўрЃА рЃЊрЃљ рЃќрЃЮрЃњрЃљрЃЊрЃљрЃЊ рЃЏрЃљрЃарЃерЃарЃБрЃбрЃўрЃАрЃЌрЃЋрЃўрЃА. рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃЮрЃЏрЃЉрЃўрЃюрЃљрЃфрЃўрЃљ:

рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃў рЃцрЃћрЃарЃћрЃЉрЃў (Monochromatic): рЃћрЃарЃЌрЃў рЃцрЃћрЃарЃўрЃА рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃдрЃўрЃљ рЃЊрЃљ рЃЏрЃБрЃЦрЃў рЃџрЃБрЃарЃ»рЃў рЃбрЃЮрЃюрЃћрЃЉрЃў.

рЃЏрЃарЃљрЃЋрЃљрЃџрЃцрЃћрЃарЃЮрЃЋрЃљрЃюрЃў рЃцрЃћрЃарЃћрЃЉрЃў (Complementary): рЃЮрЃарЃў рЃцрЃћрЃарЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃћрЃарЃЌрЃЏ druhрЃћрЃџрЃА рЃћрЃгрЃўрЃюрЃљрЃљрЃдрЃЏрЃЊрЃћрЃњрЃћрЃЉрЃўрЃљрЃю рЃЋрЃљрЃарЃЊрЃюрЃўрЃА рЃгрЃарЃћрЃќрЃћ.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃгрЃўрЃЌрЃћрЃџрЃў рЃЊрЃљ рЃЏрЃгрЃЋрЃљрЃюрЃћ.

рЃљрЃюрЃљрЃџрЃЮрЃњрЃўрЃБрЃарЃў рЃцрЃћрЃарЃћрЃЉрЃў (Analogous): рЃАрЃљрЃЏрЃў рЃцрЃћрЃарЃў, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃўрЃЌ рЃљрЃарЃўрЃљрЃю рЃцрЃћрЃарЃўрЃА рЃгрЃарЃћрЃерЃў.

рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ, рЃџрЃБрЃарЃ»рЃў, рЃфрЃўрЃАрЃцрЃћрЃарЃў рЃЊрЃљ рЃЏрЃгрЃЋрЃљрЃюрЃћ.

РюЁ рЃцрЃћрЃарЃўрЃА рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљ рЃЊрЃљ рЃћрЃЏрЃЮрЃфрЃўрЃБрЃарЃў рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ
рЃцрЃћрЃарЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃњрЃљрЃЋрЃџрЃћрЃюрЃљрЃА рЃљрЃ«рЃЊрЃћрЃюрЃЊрЃћрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃюрЃгрЃДрЃЮрЃЉрЃљрЃќрЃћ рЃЊрЃљ рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃЌрЃћрЃЏрЃљ рЃњрЃљрЃљрЃцрЃЮрЃарЃЏрЃЮрЃА.

рЃцрЃћрЃарЃў	рЃћрЃЏрЃЮрЃфрЃўрЃБрЃарЃў рЃњрЃљрЃЋрЃџрЃћрЃюрЃљ	рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ
рЃгрЃўрЃЌрЃћрЃџрЃў	рЃћрЃюрЃћрЃарЃњрЃўрЃљ, рЃЋрЃюрЃћрЃЉрЃљ, рЃЊрЃљрЃФрЃљрЃЉрЃБрЃџрЃЮрЃЉрЃљ	рЃАрЃгрЃарЃљрЃцрЃў рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃљ (рЃдрЃўрЃџрЃљрЃЎрЃћрЃЉрЃў, рЃњрЃљрЃЏрЃљрЃцрЃарЃЌрЃ«рЃўрЃџрЃћрЃЉрЃћрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃћрЃЉрЃў)
рЃџрЃБрЃарЃ»рЃў	рЃюрЃЊрЃЮрЃЉрЃљ, рЃЏрЃерЃЋрЃўрЃЊрЃЮрЃЉрЃљ, рЃАрЃбрЃљрЃЉрЃўрЃџрЃБрЃарЃЮрЃЉрЃљ	рЃърЃарЃЮрЃцрЃћрЃАрЃўрЃЮрЃюрЃљрЃџрЃБрЃарЃў рЃАрЃљрЃўрЃбрЃћрЃЉрЃў, рЃЉрЃљрЃюрЃЎрЃћрЃЉрЃў
рЃЏрЃгрЃЋрЃљрЃюрЃћ	рЃЏрЃерЃЋрЃўрЃЊрЃЮрЃЉрЃљ, рЃЉрЃБрЃюрЃћрЃЉрЃљ, рЃАрЃўрЃЏрЃерЃЋрЃўрЃЊрЃћ	рЃћрЃЎрЃЮрЃџрЃЮрЃњрЃўрЃБрЃарЃў, рЃ»рЃљрЃюрЃЏрЃарЃЌрЃћрЃџрЃЮрЃЉрЃўрЃА рЃАрЃљрЃўрЃбрЃћрЃЉрЃў
рЃДрЃЋрЃўрЃЌрЃћрЃџрЃў	рЃАрЃўрЃ«рЃљрЃарЃБрЃџрЃў, рЃДрЃБрЃарЃљрЃЊрЃдрЃћрЃЉрЃљ	рЃАрЃљрЃњрЃљрЃюрЃњрЃћрЃЉрЃЮ рЃерЃћрЃбрЃДрЃЮрЃЉрЃўрЃюрЃћрЃЉрЃћрЃЉрЃў, рЃЉрЃљрЃЋрЃерЃЋрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃњрЃљрЃюрЃЎрЃБрЃЌрЃЋрЃюрЃўрЃџрЃў рЃАрЃљрЃўрЃбрЃћрЃЉрЃў
рЃерЃљрЃЋрЃў	рЃћрЃџрЃћрЃњрЃљрЃюрЃбрЃБрЃарЃЮрЃЉрЃљ, рЃАрЃћрЃарЃўрЃЮрЃќрЃБрЃџрЃЮрЃЉрЃљ	рЃЏрЃЮрЃЊрЃљ, рЃЏрЃЊрЃўрЃЊрЃарЃБрЃџрЃў рЃЉрЃарЃћрЃюрЃЊрЃћрЃЉрЃў

РюЁ CSS рЃцрЃћрЃарЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃћрЃЉрЃў
css
Copy
 
/* рЃцрЃЮрЃюрЃў рЃЊрЃљ рЃбрЃћрЃЦрЃАрЃбрЃўрЃА рЃцрЃћрЃарЃў */
body {
  background-color: #f0f0f0; /* рЃдрЃўрЃљ рЃюрЃљрЃфрЃарЃўрЃАрЃцрЃћрЃарЃў рЃцрЃЮрЃюрЃў */
  color: #333333; /* рЃЏрЃБрЃЦрЃў рЃбрЃћрЃЦрЃАрЃбрЃў */
}

/* рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃцрЃћрЃарЃў */
button {
  background-color: #3498db; /* рЃфрЃўрЃАрЃцрЃћрЃарЃў рЃдрЃўрЃџрЃљрЃЎрЃў */
  color: white;
  border: none;
  padding: 10px 20px;
  border-radius: 5px;
  cursor: pointer;
}

  background-color: #2980b9; /* рЃдрЃўрЃџрЃљрЃЎрЃўрЃА рЃцрЃћрЃарЃў рЃњрЃљрЃЊрЃљрЃцрЃЮрЃарЃЏрЃўрЃарЃћрЃЉрЃўрЃАрЃљрЃА */
}
­Ъћџ рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ:
рЃцрЃћрЃарЃћрЃЉрЃў CSS-рЃерЃў рЃФрЃљрЃџрЃўрЃљрЃю рЃФрЃџрЃўрЃћрЃарЃў рЃўрЃюрЃАрЃбрЃарЃБрЃЏрЃћрЃюрЃбрЃўрЃљ, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃерЃћрЃБрЃФрЃџрЃўрЃљ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃљрЃцрЃЮрЃарЃЏрЃЮрЃА рЃњрЃЋрЃћрЃарЃЊрЃў рЃЊрЃљ рЃњрЃљрЃБрЃЏрЃ»рЃЮрЃЉрЃћрЃАрЃЮрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџрЃћрЃЉрЃљ. рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃцрЃћрЃарЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (HEX, RGB, HSL) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃњрЃљрЃФрЃџрЃћрЃЋрЃЌ рЃАрЃўрЃќрЃБрЃАрЃбрЃўрЃЌ рЃљрЃЎрЃЮрЃюрЃбрЃарЃЮрЃџрЃЮрЃЌ рЃцрЃћрЃарЃўрЃА рЃбрЃЮрЃюрЃћрЃЉрЃў, рЃюрЃљрЃЌрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃГрЃЋрЃўрЃарЃЋрЃљрЃџрЃЮрЃЉрЃљ.



 

Dimensions
 
­Ъје CSS-рЃўрЃА рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў (Dimensions)
CSS-рЃерЃў рЃњрЃљрЃюрЃќрЃЮрЃЏрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃўрЃА, рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА, рЃЏрЃљрЃарЃЪрЃћрЃЉрЃўрЃА, рЃАрЃљрЃќрЃдрЃЋрЃарЃћрЃЉрЃўрЃА рЃЊрЃљ рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃўрЃА рЃќрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃЊрЃњрЃћрЃюрЃљрЃЊ. рЃљрЃарЃАрЃћрЃЉрЃЮрЃЉрЃА рЃарЃљрЃЏрЃЊрЃћрЃюрЃўрЃЏрЃћ рЃЏрЃћрЃЌрЃЮрЃЊрЃў, рЃарЃЮрЃЏрЃџрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌрЃљрЃф рЃерЃћрЃњрЃЋрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃЮрЃЌ рЃќрЃЮрЃЏрЃћрЃЉрЃў.

РюЁ рЃќрЃЮрЃЏрЃўрЃА рЃћрЃарЃЌрЃћрЃБрЃџрЃћрЃЉрЃў
рЃърЃўрЃЦрЃАрЃћрЃџрЃў (px)

рЃърЃўрЃЦрЃАрЃћрЃџрЃў рЃљрЃарЃўрЃА рЃАрЃбрЃљрЃбрЃўрЃЎрЃБрЃарЃў рЃћрЃарЃЌрЃћрЃБрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃА рЃърЃўрЃарЃЊрЃљрЃърЃўрЃа рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљрЃА рЃћрЃЎрЃарЃљрЃюрЃќрЃћ.

рЃЏрЃљрЃА рЃ«рЃерЃўрЃарЃљрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃћрЃю рЃЎрЃЮрЃюрЃЎрЃарЃћрЃбрЃБрЃџрЃў рЃќрЃЮрЃЏрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃћрЃЋрЃљрЃЊ.

css
 
div {
  width: 200px;
  height: 100px;
}
рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃў (%)

рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃБрЃџрЃў рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЏрЃўрЃЌрЃўрЃЌрЃћрЃЉрЃћрЃю рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљрЃА рЃЏрЃўрЃАрЃў рЃЏрЃерЃЮрЃЉрЃџрЃўрЃБрЃарЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА рЃЏрЃўрЃ«рЃћрЃЊрЃЋрЃўрЃЌ.

рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЏрЃљрЃерЃўрЃю, рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃАрЃўрЃњрЃљрЃюрЃўрЃАрЃљ рЃЊрЃљ рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА рЃЏрЃЮрЃарЃњрЃћрЃЉрЃљ рЃљрЃБрЃфрЃўрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ.

css
 
div {
  width: 50%; /* рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃћ рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў рЃЎрЃЮрЃюрЃбрЃћрЃўрЃюрЃћрЃарЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃўрЃА 50%-рЃўрЃљ */
}
View Width (vw) рЃЊрЃљ View Height (vh)

vw (view width) Рђћ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃў.

vh (view height) Рђћ рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃюрЃбрЃў.

css
 
div {
  width: 50vw;  /* рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃњрЃљрЃюрЃўрЃА 50% */
  height: 50vh; /* рЃћрЃЎрЃарЃљрЃюрЃўрЃА рЃАрЃўрЃЏрЃљрЃдрЃџрЃўрЃА 50% */
}
rem (root em)

rem рЃљрЃарЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃцрЃЮрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА рЃЏрЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃўрЃА рЃћрЃарЃЌрЃћрЃБрЃџрЃў. рЃЌрЃБ HTML рЃцрЃљрЃўрЃџрЃерЃў рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ рЃцрЃЮрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ 16pxрЃљ, рЃЏрЃљрЃерЃўрЃю 1rem = 16px.

css
 
div {
  width: 10rem; /* 10 * 16px = 160px */
  height: 5rem; /* 5 * 16px = 80px */
}
em

em рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЉрЃљрЃЋрЃерЃЋрЃўрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА, рЃарЃљрЃЌрЃљ рЃњрЃљрЃЏрЃЮрЃ«рЃљрЃбрЃЮрЃА рЃќрЃЮрЃЏрЃљ рЃЏрЃерЃЮрЃЉрЃћрЃџ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃЌрЃљрЃю рЃЏрЃўрЃЏрЃљрЃарЃЌрЃћрЃЉрЃљрЃерЃў.

css
 
div {
  font-size: 2em; /* рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃцрЃЮрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА 2x */
  width: 5em;     /* рЃЏрЃерЃЮрЃЉрЃћрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃўрЃА 5x */
}
РюЁ max-width, min-width, max-height, min-height
CSS-рЃерЃў рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ max-width рЃЊрЃљ min-width -рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃарЃљрЃЌрЃљ рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃЮрЃЌ рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃљрЃю рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃарЃў рЃќрЃЮрЃЏрЃљ.

max-width: рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃа рЃАрЃўрЃњрЃљрЃюрЃћрЃА.

min-width: рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃа рЃАрЃўрЃњрЃљрЃюрЃћрЃА.

max-height: рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃљрЃЦрЃАрЃўрЃЏрЃљрЃџрЃБрЃа рЃАрЃўрЃЏрЃљрЃдрЃџрЃћрЃА.

min-height: рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃЏрЃўрЃюрЃўрЃЏрЃљрЃџрЃБрЃа рЃАрЃўрЃЏрЃљрЃдрЃџрЃћрЃА.

css
 
div {
  max-width: 500px;
  min-height: 200px;
  width: 80%; /* рЃцрЃљрЃарЃЌрЃЮ рЃўрЃЦрЃюрЃћрЃЉрЃљ 80% */
  background-color: lightpink;
}
рЃљрЃЏ рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, div рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ 500px-рЃќрЃћ рЃцрЃљрЃарЃЌрЃЮ рЃЊрЃљ рЃЏрЃўрЃАрЃў рЃАрЃўрЃЏрЃљрЃдрЃџрЃћ рЃљрЃа рЃўрЃЦрЃюрЃћрЃЉрЃљ 200px-рЃќрЃћ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃў.

РюЁ Box-sizing
box-sizing рЃљрЃарЃўрЃА CSS рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃљрЃЋрЃА, рЃЌрЃБ рЃарЃЮрЃњрЃЮрЃа рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ.

content-box: рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃЏрЃ«рЃЮрЃџрЃЮрЃЊ рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃќрЃћ (content), рЃљрЃа рЃўрЃЌрЃЋрЃџрЃўрЃА padding-рЃАрЃљ рЃЊрЃљ border-рЃА.

border-box: рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃќрЃЮрЃЏрЃљ рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃљрЃЊ, рЃерЃўрЃњрЃЌрЃљрЃЋрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ, padding рЃЊрЃљ border-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ.

css
 
div {
  width: 200px;
  padding: 20px;
  border: 5px solid black;
  box-sizing: border-box;  /* рЃќрЃЮрЃЏрЃљ рЃўрЃЌрЃЋрЃџрЃћрЃЉрЃљ рЃАрЃарЃБрЃџрЃљрЃЊ */
}
рЃарЃЮрЃЊрЃћрЃАрЃљрЃф рЃўрЃДрЃћрЃюрЃћрЃЉ box-sizing: border-box, рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃћрЃарЃЌрЃЮ рЃќрЃЮрЃЏрЃљ рЃўрЃЦрЃюрЃћрЃЉрЃљ 200px, padding рЃЊрЃљ border-рЃўрЃА рЃЕрЃљрЃЌрЃЋрЃџрЃўрЃЌ.

РюЁ Flexbox
Flexbox рЃљрЃарЃўрЃА рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЋрЃљ рЃБрЃцрЃарЃЮ рЃЏрЃЮрЃЦрЃюрЃўрЃџрЃў рЃЊрЃљ рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃбрЃўрЃБрЃџрЃў рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ.

justify-content: рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃўрЃА рЃЊрЃљрЃАрЃљрЃгрЃДрЃўрЃАрЃў, рЃЊрЃљрЃАрЃљрЃАрЃарЃБрЃџрЃў рЃЊрЃљ рЃфрЃћрЃюрЃбрЃарЃў.

align-items: рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃўрЃА рЃњрЃљрЃюрЃџрЃљрЃњрЃћрЃЉрЃљ рЃЌрЃўрЃЌрЃЮрЃћрЃБрЃџ рЃдрЃћрЃарЃФрЃќрЃћ.

css
 
.container {
  display: flex;
  justify-content: space-between;
  width: 100%;
}
