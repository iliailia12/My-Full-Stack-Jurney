­Ъћг 1. рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃцрЃўрЃџрЃЮрЃАрЃЮрЃцрЃўрЃљ РђЊ рЃарЃљрЃбрЃЮрЃЏ "рЃАрЃЎрЃарЃўрЃърЃбрЃў"?
рЃАрЃўрЃбрЃДрЃЋрЃљ "рЃАрЃЎрЃарЃўрЃърЃбрЃў" рЃЏрЃЮрЃЊрЃўрЃА рЃЌрЃћрЃљрЃбрЃарЃљрЃџрЃБрЃарЃў рЃљрЃюрЃЎрЃћрЃбрЃўрЃЊрЃљрЃю РђЊ рЃАрЃфрЃћрЃюрЃљрЃарЃўрЃЋрЃўрЃЌрЃљрЃљ, рЃАрЃљрЃЊрЃљрЃф рЃќрЃБрЃАрЃбрЃљрЃЊ рЃгрЃћрЃарЃўрЃљ рЃарЃљ, рЃарЃЮрЃЊрЃўрЃА, рЃарЃЮрЃњрЃЮрЃа рЃБрЃюрЃЊрЃљ рЃерЃћрЃАрЃарЃБрЃџрЃЊрЃћрЃА. рЃбрЃћрЃЦрЃюрЃўрЃЎрЃљрЃерЃўрЃљрЃф рЃўрЃњрЃўрЃЋрЃћ рЃърЃарЃўрЃюрЃфрЃўрЃърЃўрЃљ:
Script = рЃАрЃфрЃћрЃюрЃљрЃарЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃА рЃљрЃю рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃА рЃљрЃФрЃџрЃћрЃЋрЃА рЃќрЃБрЃАрЃб рЃЏрЃЮрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃћрЃЉрЃА рЃерЃћрЃАрЃљрЃАрЃарЃБрЃџрЃћрЃЉрЃџрЃљрЃЊ.

РџЎ№ИЈ 2. рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА Script рЃЎрЃЮрЃЏрЃърЃўрЃБрЃбрЃћрЃарЃерЃў
рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃА рЃБрЃЏрЃћрЃбрЃћрЃАрЃЮрЃЉрЃљ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃфрЃўрЃўрЃЌ (рЃљрЃа рЃЎрЃЮрЃЏрЃърЃўрЃџрЃўрЃарЃЊрЃћрЃЉрЃљ, рЃарЃЮрЃњрЃЮрЃарЃф C рЃљрЃю Java), рЃарЃљрЃф рЃюрЃўрЃерЃюрЃљрЃЋрЃА:

­ЪЊѓ рЃерЃћрЃю рЃЦрЃЏрЃюрЃў рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃцрЃљрЃўрЃџрЃА (script.js, script.py, myscript.sh)

РџЎ№ИЈ рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃљрЃю рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃў рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЏрЃљрЃА рЃ«рЃљрЃќрЃўрЃЌ рЃ«рЃљрЃќрЃќрЃћ

­ЪДа рЃљрЃа рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃў рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ Рєњ рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА "рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў"

­ЪЊЅ рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃюрЃљрЃЎрЃџрЃћрЃЉрЃљрЃЊ рЃЮрЃърЃбрЃўрЃЏрЃўрЃќрЃўрЃарЃћрЃЉрЃБрЃџрЃА, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃЊрЃА

­ЪДа 3. рЃАрЃЎрЃарЃўрЃърЃбрЃў vs рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ Рђћ рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃљ
рЃАрЃЎрЃарЃўрЃърЃбрЃў	рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ
рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃўрЃарЃћрЃЉрЃБрЃџрЃў	рЃЎрЃЮрЃЏрЃърЃўрЃџрЃўрЃарЃћрЃЉрЃБрЃџрЃў
рЃАрЃгрЃарЃљрЃцрЃљрЃЊ рЃўрЃгрЃћрЃарЃћрЃЉрЃљ	рЃ«рЃерЃўрЃарЃљрЃЊ рЃАрЃљрЃГрЃўрЃарЃЮрЃћрЃЉрЃА рЃЊрЃљрЃњрЃћрЃњрЃЏрЃЋрЃљрЃА
рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ рЃарЃћрЃљрЃџрЃБрЃа рЃЊрЃарЃЮрЃерЃў	рЃЎрЃЮрЃЏрЃърЃўрЃџрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃЊрЃљрЃбрЃћрЃАрЃбрЃЋрЃљ рЃАрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ
рЃ«рЃерЃўрЃарЃљрЃЊ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃћрЃюрЃћрЃЉрЃА	рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА рЃЊрЃљрЃЉрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА (C, Rust...)

рЃЏрЃљрЃњ:
JavaScript, Python, Lua, Bash = рЃАрЃЎрЃарЃўрЃърЃбрЃБрЃџрЃў рЃћрЃюрЃћрЃЉрЃў
C++, Rust, Go = рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃБрЃџрЃў рЃћрЃюрЃћрЃЉрЃў

­ЪњА 4. рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃарЃЮрЃџрЃў рЃЌрЃљрЃюрЃљрЃЏрЃћрЃЊрЃарЃЮрЃЋрЃћ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃерЃў
­Ъњ╗ Web:
JavaScript рЃЏрЃљрЃарЃЌрЃљрЃЋрЃА рЃЋрЃћрЃЉрЃАрЃљрЃўрЃбрЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА, рЃўрЃюрЃбрЃћрЃарЃљрЃЦрЃфрЃўрЃљрЃА, рЃљрЃюрЃўрЃЏрЃљрЃфрЃўрЃћрЃЉрЃА, рЃљрЃАрЃўрЃюрЃЦрЃарЃЮрЃюрЃБрЃџ рЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЋрЃћрЃЉрЃА.

React, Vue, AngularРђћall driven by script-based logic

РџЎ№ИЈ DevOps / System Admin:
Bash, Shell, PowerShell рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃЌ рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃўрЃќрЃЊрЃћрЃЉрЃљ рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ, рЃЉрЃћрЃЦрЃљрЃцрЃў, рЃЏрЃЮрЃюрЃўрЃбрЃЮрЃарЃўрЃюрЃњрЃў.

­ЪДф рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃљрЃюрЃљрЃџрЃўрЃќрЃў:
Python-рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃњрЃљрЃгрЃЏрЃћрЃюрЃЊрЃљ, рЃљрЃюрЃљрЃџрЃўрЃќрЃў, рЃЋрЃўрЃќрЃБрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљ.

­Ъцќ рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃџрЃћрЃЦрЃбрЃў:
рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃўрЃЌ рЃ«рЃЊрЃћрЃЉрЃљ рЃЏрЃЮрЃЊрЃћрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃгрЃЋрЃарЃЌрЃюрЃљ, рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ, API-рЃЊрЃљрЃю рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃўрЃА рЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљ.

­ЪћЇ 5. рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃўрЃА рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃЊрЃћрЃбрЃљрЃџрЃћрЃЉрЃў (JavaScript-рЃўрЃА рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў)
рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА HTML-рЃА

рЃгрЃљрЃљрЃгрЃДрЃЊрЃћрЃЉрЃљ <script> рЃбрЃћрЃњрЃА

рЃњрЃљрЃЊрЃљрЃћрЃфрЃћрЃЏрЃљ JavaScript рЃўрЃюрЃбрЃћрЃарЃърЃарЃћрЃбрЃљрЃбрЃЮрЃарЃА (рЃЏрЃљрЃњ. V8 Chrome-рЃерЃў)

рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ Execution Context РђЊ рЃњрЃљрЃарЃћрЃЏрЃЮ, рЃАрЃљрЃЊрЃљрЃф рЃЎрЃЮрЃЊрЃў рЃњрЃљрЃўрЃерЃЋрЃћрЃЉрЃљ

рЃ«рЃЊрЃћрЃЉрЃљ рЃЎрЃЮрЃЊрЃўрЃА рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃљ (рЃЏрЃћрЃ«рЃАрЃўрЃћрЃарЃћрЃЉрЃљрЃерЃў "stack"-рЃќрЃћ рЃ«рЃЊрЃћрЃЉрЃљ рЃЮрЃърЃћрЃарЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ)

Execution Stack рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃў:

­ЪЉЅ Stack:

­ЪД▒ 6. рЃАрЃЎрЃарЃўрЃърЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў
рЃДрЃЋрЃћрЃџрЃљ рЃАрЃЎрЃарЃўрЃърЃбрЃў (рЃћрЃюрЃўрЃЊрЃљрЃю рЃЊрЃљрЃЏрЃЮрЃБрЃЎрЃўрЃЊрЃћрЃЉрЃџрЃљрЃЊ) рЃ«рЃерЃўрЃарЃљрЃЊ рЃерЃћрЃўрЃфрЃљрЃЋрЃА:

рЃфрЃЋрЃџрЃљрЃЊрЃћрЃЉрЃў (Variables): let x = 5

рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў (Functions): function greet() { ... }

рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃў: рЃЏрЃљрЃАрЃўрЃЋрЃћрЃЉрЃў, рЃЮрЃЉрЃўрЃћрЃЦрЃбрЃћрЃЉрЃў

рЃърЃўрЃарЃЮрЃЉрЃћрЃЉрЃў (Conditions): if, else

рЃерЃћрЃбрЃљрЃюрЃўрЃА/рЃњрЃљрЃЏрЃЮрЃбрЃљрЃюрЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў: alert(), print(), echo...

­ЪДф 7. рЃАрЃЎрЃарЃўрЃърЃбрЃћрЃЉрЃў рЃерЃћрЃўрЃФрЃџрЃћрЃЉрЃљ рЃўрЃДрЃЮрЃА:

Client-side	рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃЏрЃЮрЃгрЃДрЃЮрЃЉрЃўрЃџрЃЮрЃЉрЃљрЃќрЃћ (рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃерЃў)	JavaScript
